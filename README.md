# 🚀 Plataforma BI de Reporting Corporativo

## 📌 Problema
Muchas organizaciones carecen de visibilidad sobre el rendimiento operativo, especialmente en relación con retrasos en pedidos, desempeño de proveedores y evolución de ingresos.

## 🎯 Objetivo
El objetivo de este proyecto es diseñar e implementar una solución de Business Intelligence utilizando Power BI y Report Builder para mejorar la toma de decisiones basada en datos.

## 🧠 Alcance
- Modelado de datos en esquema estrella  
- Definición de KPIs de negocio  
- Desarrollo de dashboards interactivos  
- Creación de informes paginados  

## 🛠️ Tecnologías
- Power BI  
- DAX  
- Report Builder  
- Python  

## 📊 KPIs
- Total de pedidos  
- Ingresos totales  
- % pedidos retrasados  
- Promedio de días de retraso
